1. Introduction to Python

Python is a high-level, interpreted, and general-purpose programming language.
It emphasizes readability and simplicity.

Key Features:

Easy to learn and read

Interpreted (no need for compilation) .

Dynamically typed (no need to declare variable types)

Supports Object-Oriented, Functional, and Procedural programming

Huge standard library

Portable (works on multiple platforms)

12. File Handling



| Mode   | Meaning            | Description                                                                |
| ------ | ------------------ | -------------------------------------------------------------------------- |
| `'r'`  | Read               | Opens the file for reading (default). Error if the file doesn’t exist.     |
| `'w'`  | Write              | Opens the file for writing. Creates a new file or overwrites if it exists. |
| `'a'`  | Append             | Opens the file for writing but adds to the end (does not overwrite).       |
| `'x'`  | Exclusive creation | Creates a new file but gives an error if it already exists.                |
| `'r+'` | Read & Write       | Opens the file for both reading and writing. Must exist.                   |
| `'w+'` | Write & Read       | Overwrites existing file or creates a new one.                             |
| `'a+'` | Append & Read      | Opens file for appending and reading; creates new file if needed.          |
| `'b'`  | Binary mode        | Used for binary files like images, videos, etc. (e.g., `'rb'`, `'wb'`).    |
| `'t'`  | Text mode          | Default mode for text files (e.g., `'rt'`, `'wt'`).                        |

# Write to file
f = open("data.txt", "w")
f.write("Hello Python")
f.close()

# Read from file
f = open("data.txt", "r")
print(f.read())
f.close()

13. Exception Handling
try:
    num = int(input("Enter number: "))
    print(10 / num)
except ZeroDivisionError:
    print("Cannot divide by zero!")
except ValueError:
    print("Invalid input")
finally:
    print("Program ended")



14. Object-Oriented Programming (OOP)
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        print("Hello, my name is", self.name)

p1 = Person("Alice", 25)
p1.greet()


Concepts:
Class & Object
Constructor (__init__)
Inheritance
Polymorphism
Encapsulation
