1️⃣ Division by Zero
Write a program that asks the user for two numbers and divides them.
Handle the error if the user tries to divide by zero.

2️⃣ Invalid Integer Input
Write a program that asks the user to enter an integer.
Handle the error if the user enters a non-integer value.


3️⃣ List Index Error
Given a list [10, 20, 30], ask the user to enter an index and print the element at that index.
Handle the error if the user enters an invalid index.


4️⃣ File Not Found
Write a program to open and read a file called data.txt.
Handle the error if the file does not exist.


5️⃣ Multiple Exceptions
Write a program that asks the user for two numbers and divides them.
Handle both the error for invalid input (non-integer) and division by zero.
